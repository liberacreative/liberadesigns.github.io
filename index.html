Publication Design: Library Newsletters

Layout / Image Manipulation & Selection / Images drawn from featured Library artists

The Problem

  • I was tasked with formatting City of Sydney Library's 3 bi-monthly newsletter editions for a contracted period.
  • Each newsletter had to accommodate for shifting, often lengthy amounts of text and imagery, as well as engage relevant demographics with respect to the City of Sydney style guide.

The Solution

  • Meticulous attention was given to space, alignment and typesetting to accommodate for shifting amounts of content within each newsletter.
  • Graphical icons (e.g. calendar legend, computer skills competency rating) were moreover utilised when relevant, to simplify layout and reduce textual clutter.

Library Posters: Adults

Layout / Image Manipulation & Selection / Images drawn from Library stock photo archives

The Problem

  • For a contracted period, I was tasked with formatting the majority of City of Sydney Library's event posters to be displayed at Library branches.
  • Such posters were produced in high-volume, with the Library needing approximately 20-40 different posters per month to promote events for Adults, Youth and Late Night Library.

The Solution

  • In keeping with the high-volume nature of this project, I introduced a grid (i.e. structured type columns) to accommodate for large amounts of variable content.
  • The grid essentially allowed for different amounts of event text and imagery to be slotted in quickly and consistently, in effect, maintaining the same professional structure for each poster.

The Solution
